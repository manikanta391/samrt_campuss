# Smart Campus Management System

## Project Description

Smart Campus Management System is a Java and MySQL based application developed to manage campus activities efficiently.

## Features

* Student Management
* Faculty Management
* Attendance Management
* Complaint Management
* Event Management
* User Login System

## Technologies Used

* Java
* JDBC
* MySQL

## How to Run

1. Install Java JDK
2. Install MySQL
3. Create the required database and tables
4. Add MySQL Connector JAR
5. Compile and run MainMenu.java
